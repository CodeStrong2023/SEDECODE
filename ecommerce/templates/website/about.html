Acerca de Nuestro Proyecto

Proyecto Integrador de la UTN San Rafael

Este es un proyecto integrador de la Universidad Tecnológica Nacional (UTN), sede San Rafael. El proyecto consiste en un ecommerce de venta de comida francesa, donde los usuarios pueden explorar una variedad de platillos franceses y realizar compras de manera sencilla y segura.

Para facilitar el proceso de pago, hemos integrado Mercado Pago, lo que permite a nuestros clientes realizar transacciones de forma rápida y confiable.

Herramientas Utilizadas

Las herramientas utilizadas en el desarrollo de este proyecto incluyen:

  • HTML: Para la estructura de las páginas web.
  • CSS: Para el diseño y estilo visual del sitio.
  • JavaScript: Para mejorar la interactividad y experiencia del usuario.
  • Flask: Un micro framework de Python que facilita la creación de aplicaciones web.
  • PostgreSQL: Para el almacenamiento y gestión de datos en una base de datos relacional.
